SQL

EXEC sp_recompile 'YourSproc'
It is a SQL Server command that forces the recompilation of a stored procedure. It discards the existing execution plan and generates a new plan when the procedure is next executed. This can enhance performance by considering the current database state, but should be used sparingly due to potential impact on system performance.
